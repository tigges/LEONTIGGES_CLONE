# LEONTIGGES_CLONE

Static mirror of [leontigges.com](https://leontigges.com/) lives in **`leontigges-site/`**.

## View on GitHub Pages

If the site looks like only this readme, **Pages is pointed at the wrong folder**. Either:

1. **Settings → Pages → Branch `gh-pages`**, folder `/ (root)` — full site at  
   `https://<user>.github.io/LEONTIGGES_CLONE/`, or  
2. Keep **Branch `main`**, folder `/ (root)` — the root `index.html` here redirects to **`leontigges-site/`**.
